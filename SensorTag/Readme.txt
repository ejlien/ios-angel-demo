Texas Instruments Inc.SensorTag Examples source codeRelease Notes--------------------------------------------------------------------------------------------------------------------------------------------------------------Version 1.1.0October  22, 2013Release updated to support iOS7.This installer contains two iOS source codes examples that can be compiled and used to communicate with CC2541DK-Sensor (SensorTag), www.ti.com/sensortag* Sensor Example Source Code 1.1.0 is an app that will communicate with all the different sensors on the SensorTag design.* Over the Air Download  Example Source Code 1.1.0 is an app that allows for over the air download of new Firmware to the SensorTagFor technical support and more information, please see the SensorTag product page (www.ti.com/sensortag), the SensorTag wiki page (www.ti.com/sensortag-wiki) or visit Texas Instruments Bluetooth low energy E2E Forum:http://e2e.ti.com/support/low_power_rf/f/538.aspx--------------------------------------------------------------------------------------------------------------------------------------------------------------
